## Project Overview
This repository contains the files and documentation for my Data Analytics Capstone project focusing on EmporiUm, a virtual student bookstore. The objective of this project is to clean, model, analyze, and visualize four years of business-related sales data using Power BI to support better business decision-making. 

## Key Insights & Highlights
* **The North Hartford Domination:** A deep dive into the operational efficiency and massive transaction volume of the North Hartford store, which accounts for the vast majority of the total company sales share.
* **Macro-Level Sales Trends:** Comprehensive analysis of overall sales performance and trends over the four-year period.
* **Regional Top Sellers:** Identification of the top-selling books (excluding textbooks) tailored to regional preferences to drive targeted marketing.

## Power BI Report Features
The interactive Power BI report is divided into multiple pages and features:
* A line chart showing overall sales trends over time.
* A column/bar chart displaying sales performance by product category.
* A donut chart illustrating the relative sales share for each state in the region.
* A detailed table highlighting the top-selling books and their authors.
